# 🌐 LinguaFlow — AI Language Translator

AI-powered Language Translation Web App built with Flask & Google Translate API.

## ✨ Features
- 🌍 100+ Languages | 🎨 Glassmorphism UI | 🌙 Dark Mode
- 🔊 Text-to-Speech | 📋 Copy | 📥 Download Translation
- 🔄 Language Swap | 🕒 History | 📱 Mobile Responsive

## 🛠️ Tech Stack
Python • Flask • deep-translator • HTML • CSS • JavaScript

## 🚀 Run Locally
```bash
pip install -r requirements.txt
